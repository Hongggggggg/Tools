此脚本为串口示波器，目的是将串口数据波形通过UI界面显示出来，可直接运行.exe文件，无需搭建其他环境



使用方法：

- 通过UI界面设置串口参数
- 设置串口输出变量名称，串口输出格式需为 “变量名称 + ... + 变量值\r\n”, ...代表任意内容
  - 例：lumi：10\r\n
- 设置图形界面参数
  - Y轴上下限
  - 波形点数
- 设置完毕后点击Start即可
- 点击Pause可暂停波形输出，Start可继续运行
